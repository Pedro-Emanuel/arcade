# L0 - @par ou impar

![Imagem eh Positivo](cover.jpg)

## Ação

Implemente um programa que recebe um número inteiro e diga se ele é par ou impar.

### Entrada

- Um inteiro

### Saída

- "PAR" se o número for impar
- "IMPAR" se o número for par

## Exemplos

```txt
>>>>>>>> 0
3
========
IMPAR
<<<<<<<<

>>>>>>>> 1
12
========
PAR
<<<<<<<<

>>>>>>>> 2
33
========
IMPAR
<<<<<<<<

>>>>>>>> 3
0
========
PAR
<<<<<<<<

>>>>>>>> 4
7
========
IMPAR
<<<<<<<<

>>>>>>>> 5
20
========
PAR
<<<<<<<<

>>>>>>>> 6
15
========
IMPAR
<<<<<<<<
```
