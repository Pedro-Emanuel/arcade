# L0 - @maximo entre dois números

![Imagem eh Positivo](cover.jpg)

## Ação

Implemente um programa que recebe dois números inteiros e imprime o maior.

### Entrada

- Dois inteiros, separados por espaço

### Saída

- Imprima o maior número

## Exemplos

```txt
>>>>>>>> 0
4 9
========
9
<<<<<<<<

>>>>>>>> 1
56 7
========
56
<<<<<<<<

>>>>>>>> 2
46 -789
========
46
<<<<<<<<

>>>>>>>> 3
-45 -125
========
-45
<<<<<<<<

>>>>>>>> 4
-458 -12
========
-12
<<<<<<<<

>>>>>>>> 5
458 12
========
458
<<<<<<<<

>>>>>>>> 6
458 458
========
458
<<<<<<<<

>>>>>>>> 7
0 0
========
0
<<<<<<<<

>>>>>>>> 8
-458 -458
========
-458
<<<<<<<<
```
